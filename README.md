# Radium

> A modern IDE for the 21st Century.

Radium is a powerful, hackable IDE built on [Electron](https://github.com/electron/electron), featuring:
- **Modern Design**: Beautiful leaf gradients, glows, and neon effects
- **Workshop**: Built-in marketplace for themes and plugins
- **Language Support**: Extensive support for Java, JavaScript, TypeScript, React, Python, C, C#, Assembly, C++, Rust, Ruby, and more
- **Project Support**: Full IDE experience with project management
- **Syntax Highlighting**: Advanced text highlighting for all supported languages
